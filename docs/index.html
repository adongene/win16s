Win16S：A simple pipeline to analyze 16S rRNA amplicon sequencing data on the Windows

Windows下16S rRNA基因扩增子测序数据分析的简易流程

本流程相关安装软件与数据：

1. 流程安装说明：win16s_install.txt

2. 流程使用命令说明：win16s_pipeline.txt

3. 流程测试数据：win16s_data.zip

如果你需要一步安装本流程，请下载此shell脚本,并在WSL-Ubuntu bash下运行安装：win16s_install.sh

你也可以手动下载本流程相关的安装软件包：win16s.tar.gz
